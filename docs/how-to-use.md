# How to Use These Skills

Each skill lives in its own folder under `skills/`.

Each skill contains a `SKILL.md` file with:

- a name
- a description
- trigger guidance
- review workflow
- output format

You can use these in three ways:

1. Install them into an AI agent system that supports skills.
2. Copy the `SKILL.md` content into an AI chat as a structured prompt.
3. Adapt them into internal checklists, workshop material, or governance documents.

These skills are designed to help people ask better privacy and security questions earlier.

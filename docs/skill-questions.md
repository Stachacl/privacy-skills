# Questions These Skills Help Answer

## For everyday users

- Can I paste this into ChatGPT?
- Can I upload this PDF to an AI tool?
- What should I blur in this screenshot?
- Is this form asking for too much information?
- How do I make this post safer to publish?
- Should this meeting be recorded?
- What is the minimum information I need to collect?

## For teams and organisations

- What should our staff AI use policy say?
- Which AI tools should be approved, restricted, or blocked?
- What data are we sending to vendors?
- Are we sharing more customer information than necessary?
- What should we do if data was accidentally emailed or uploaded?
- How should we review sensitive document workflows?
- What should we ask before sending data to a consultant or SaaS tool?

## For developers and architects

- Are we logging personal or confidential data?
- Are prompts and AI responses stored in logs?
- Does this API return too much data?
- Does this RAG system respect document permissions?
- Can embeddings or vector search expose sensitive documents?
- Are we using safe defaults?
- What privacy risks should we check before launch?

# NSW, Australia AI Assessment Framework

This skill is for helping users work through the New South Wales, Australia AI Assessment Framework using official NSW Government guidance and the official AIAF workbook.

It should not invent scoring, risk rules, legal obligations, or review pathways. It should guide users back to the official NSW Government AIAF resources where formal assessment is required.

## Official NSW Government resource

- [NSW, Australia AI Assessment Framework](https://www.digital.nsw.gov.au/policy/artificial-intelligence/ai-governance-assurance-and-frameworks/nsw-ai-assessment-framework)

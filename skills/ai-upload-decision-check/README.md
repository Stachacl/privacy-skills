# AI Upload Decision Check

Review content before pasting, uploading, or sending it to an AI tool, and decide whether it is safe to share, needs changes first, or should not be uploaded at all.

## When to use this

- You are about to paste a document, email, or spreadsheet into ChatGPT, Claude, Gemini, or another AI tool.
- You want to check whether a file contains personal or confidential information before uploading it.
- You are unsure whether the AI tool you are using is safe for the content you want to share.

## The checklist

1. Identify the AI tool: is it a consumer product (free tier), enterprise plan, or local/on-prem tool?
2. Check whether the content includes names, email addresses, phone numbers, or IDs.
3. Check whether the content includes customer or client information.
4. Check whether the content includes employee or HR information.
5. Check whether the content includes financial details, contracts, or pricing.
6. Check whether the content includes health, legal, or other sensitive categories.
7. Check whether the content includes internal system names, credentials, or architecture details.
8. Consider whether the content can be redacted or generalised before uploading.
9. Check whether an approved enterprise or on-prem version of the tool is available.

## Examples

- A manager wants to ask an AI to summarise a staff performance review. The document contains full names, roles, and salary information.
- A developer uploads a config file to get help with a bug. The file includes a database password.
- A teacher pastes student feedback into a consumer AI tool to summarise it. The feedback includes pupil names and learning difficulties.

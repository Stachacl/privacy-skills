# Privacy Skills

Open-source agent skills for checking privacy risks, AI data exposure, sensitive document workflows, project security, and data minimisation.

These skills help people ask better privacy and security questions before they upload, share, log, automate, or build with sensitive data.

They are practical review tools, not legal advice.

## Skills

### Everyday privacy skills

- `before-you-upload-check`
- `screenshot-privacy-check`
- `form-privacy-check`
- `safe-sharing-version`

### Team and governance skills

- `ai-data-exposure-check`
- `sensitive-document-workflow-review`
- `data-flow-mapping`
- `governance-framework-builder`

### Developer and architecture skills

- `logging-privacy-review`
- `ai-rag-privacy-review`

## Important note

These skills are educational and practical review tools. They are not legal advice, security certification, or a replacement for professional privacy, legal, or security review.

## About

Created by Berrysbay as part of an open-source effort to make privacy thinking more practical for teams building with AI, cloud tools, and sensitive data.
